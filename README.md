Creates a random sr25519 keypair -

- clone repo
- `yarn` (to install deps)
- `yarn start` (to execute)

```

via minisecret H2Nijz3bYh9FAXi7TnqhBMrppBg2Mqko7gjk6GwwDJcFLpR
```
